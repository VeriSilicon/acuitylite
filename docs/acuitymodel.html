quantize(data, quantizer='asymu8', iteration=1, minimize_layer_error=False)¶

Quantize an acuitylite network.

Usually 500~1000 pictures are needed. Do not perform quantization on acuitylite networks converted from TensorFlow Lite models that have been quantized.

Parameters

  • data – The list of inputs data generator, requires list type.

  • quantizer – The quantizer to quantize network tensors. Set from QuantizeType class, support QuantizeType.asymu8 now.

  • iteration – The number of sample image batches. requires int type.

  • minimize_layer_error – Minimize the quantization error, requires bool type. If set it will improve quantization accuracy, but take more time.

Returns
